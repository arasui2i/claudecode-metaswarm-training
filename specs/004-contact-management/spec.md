# Contact Management

## Business Goal

Manage people associated with customers.

## Backend Requirements

### Entity
Contact

Fields:
- Id
- FirstName
- LastName
- Email
- Phone
- AccountId
- Status

### APIs

- Create Contact
- Update Contact
- Get Contact
- Search Contacts
- Delete Contact

### Validation

- FirstName required
- Email required

## Frontend Requirements

### Contact List

Features:
- Search
- Add
- Edit
- Delete

Grid Columns:
- Name
- Email
- Phone

### Contact Form

Fields:
- First Name
- Last Name
- Email
- Phone
- Account

## Acceptance Criteria

- User can manage contacts
