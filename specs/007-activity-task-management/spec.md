# Activity Task Management

## Business Goal

Track customer activities and tasks.

## Backend Requirements

### Entity
Activity

Fields:
- Id
- Subject
- ActivityType
- DueDate
- Status
- AssignedTo

### APIs

- Create Activity
- Update Activity
- Get Activity
- Search Activities
- Delete Activity

### Validation

- Subject required
- DueDate required

## Frontend Requirements

### Activity List

Features:
- Search
- Add
- Edit
- Delete

Grid Columns:
- Subject
- Type
- Due Date
- Status

### Activity Form

Fields:
- Subject
- Type
- Due Date
- Status

## Acceptance Criteria

- User can manage activities
