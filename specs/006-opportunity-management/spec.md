# Opportunity Management

## Business Goal

Track sales opportunities.

## Backend Requirements

### Entity
Opportunity

Fields:
- Id
- OpportunityName
- AccountId
- ContactId
- Stage
- ExpectedRevenue
- CloseDate

### APIs

- Create Opportunity
- Update Opportunity
- Get Opportunity
- Search Opportunities
- Delete Opportunity

### Validation

- OpportunityName required
- Account required

## Frontend Requirements

### Opportunity List

Features:
- Search
- Add
- Edit
- Delete

Grid Columns:
- Opportunity
- Account
- Stage
- Revenue

### Opportunity Form

Fields:
- Name
- Account
- Contact
- Stage
- Revenue
- Close Date

## Acceptance Criteria

- User can manage opportunities
