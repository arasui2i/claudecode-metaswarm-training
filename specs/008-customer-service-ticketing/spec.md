# Customer Service Ticketing

## Business Goal

Manage customer support requests.

## Backend Requirements

### Entity
Ticket

Fields:
- Id
- TicketNumber
- Subject
- AccountId
- ContactId
- Priority
- Status

### APIs

- Create Ticket
- Update Ticket
- Get Ticket
- Search Tickets
- Delete Ticket

### Validation

- Subject required
- Priority required

## Frontend Requirements

### Ticket List

Features:
- Search
- Add
- Edit
- Delete

Grid Columns:
- Ticket Number
- Subject
- Priority
- Status

### Ticket Form

Fields:
- Subject
- Account
- Contact
- Priority
- Status

## Acceptance Criteria

- User can manage tickets
